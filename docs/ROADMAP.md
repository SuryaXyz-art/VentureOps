# Roadmap

- Add authenticated operator accounts.
- Add richer webhook event replay tooling.
- Add payment-link sharing and customer email delivery.
- Add run cancellation and pause controls.
- Add per-agent retry controls for slow LLM calls.

- Add multi-run comparison and cohort analytics.
- Add real SaaS provisioning workflows through Stripe/MCP where available.
- Add richer policy templates for industries and spend classes.
- Add approval notifications through Slack/email.
- Add persistent encrypted credential vault integration.

- Multi-business portfolio control tower.
- Agent marketplace for specialized operators.
- Runtime isolation backed by production-grade NemoClaw/OpenShell policy execution.
- Continuous P&L, risk, and fulfillment monitoring.
